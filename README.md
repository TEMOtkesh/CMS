# CMS — Web Programming Project

A full-stack CMS built with PHP (OOP), MySQL, HTML5/CSS3, and jQuery.

## Pages
- Home
- Register / Login
- User Dashboard (file upload)
- Admin Panel (user management, log viewer)
- Contact

## Stack
- **Backend:** PHP 8+ OOP, Sessions/Cookies
- **Database:** MySQL (4 tables, 1:N and N:N relations)
- **Frontend:** HTML5, CSS3 (Flexbox/Grid, animations, gradients), jQuery

## Setup
1. Import `database/schema.sql` into MySQL
2. Copy `config/db.example.php` → `config/db.php` and fill in credentials
3. Point your web server root to this folder
